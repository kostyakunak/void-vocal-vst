# Этап 1: Добавление параметра Mix ✅

## Что сделано:

1. **Упростил плагин** - убрал синтезатор и delay эффект
2. **Добавил параметр Mix** (0.0-1.0, default: 0.0)
3. **Реализовал логику Mix=0%** - звук проходит без изменений (bit-perfect dry)
4. **Упростил GUI** - только один слайдер Mix

## Ключевые изменения:

### PluginProcessor.h/cpp:
- Удален синтезатор и delay
- Добавлен параметр "mix"
- При Mix=0% звук проходит без обработки (early return)
- При Mix>0% создается wet buffer и микшируется с dry

### PluginEditor.h/cpp:
- Упрощен GUI - только слайдер Mix
- Сохранена timecode display

## Проверка:

1. **Перезапустите Ableton Live**
2. **Найдите AudioPluginDemo** в Plug-Ins
3. **Откройте плагин** - должен быть только слайдер "Mix"
4. **Проверьте Mix=0%**:
   - Добавьте аудио на трек
   - Установите Mix=0%
   - Звук должен быть идентичен оригиналу (без изменений)
